    📝 記錄今日食譜

    記錄到你自己的 Google Sheets，追蹤每日攝取量。每個家庭獨立記錄。

    🔧 首次設定教學（只需做一次）
    步驟 1：建立一個新的 Google Sheets
    步驟 2：點選「擴充功能」→「Apps Script」
    步驟 3：將以下程式碼貼入，取代原有內容：
    function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); if (sheet.getLastRow() === 0) { sheet.appendRow([ '日期', '狗狗名字', '年齡', '體重(kg)', '食材明細', '鮮食(kcal)', '飼料(kcal)', '總熱量(kcal)', '蛋白質(g)', '脂肪(g)', '碳水(g)', '纖維(g)', '備註' ]); } sheet.appendRow([ data.date, data.dogName, data.age, data.weight, data.ingredients, data.freshKcal, data.kibbleKcal, data.totalKcal, data.protein, data.fat, data.carb, data.fiber, data.note ]); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); }
    步驟 4：點「部署」→「新增部署作業」→ 類型選「網頁應用程式」
    步驟 5：「誰可以存取」改為「所有人」→ 按「部署」
    步驟 6：授權畫面按「Advanced」→「Go to ... (unsafe)」→「允許」
    步驟 7：複製產生的網址，貼到上方欄位

    完成後，每次按「記錄」就會自動新增一列到你的試算表。
